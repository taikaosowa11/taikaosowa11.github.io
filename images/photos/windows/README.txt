Put images for the "Windows" album in this folder.

Naming pattern used on this site: window_1.jpg, window_2.jpg, window_3.jpg, ...
(numbered in the order they should appear).

1. Add a cover photo named exactly:  cover.jpg
   (shown as the tile on the Photos page)

2. Add the rest of the photos as window_1.jpg, window_2.jpg, etc.
   Convert HEIC to JPG first -- most browsers can't display HEIC.

3. Open photos/windows.html and list those filenames in the
   PHOTOS array, in the order you want them to appear.
