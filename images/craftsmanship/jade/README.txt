Put images for the "Jade" craftsmanship piece in this folder.

Naming pattern used on this site: jade_1.jpg, jade_2.jpg, ...
(numbered in the order they should appear).

1. Add a cover photo named exactly:  cover.jpg
   (shown as the tile on the Craftsmanship page)

2. Add the rest of the photos as jade_1.jpg, jade_2.jpg, etc.
   Convert HEIC to JPG first -- most browsers can't display HEIC.

3. Open craftsmanship/jade.html and list those filenames in the
   PHOTOS array, in the order you want them to appear.
